## Task 6: Criar `toggleExpandFu()` (função de expandir específica da FU)

**Objetivo:** Função isolada para expandir/recolher na aba FU, evitando conflito com `toggleExpand` da aba Andamento.

```javascript
function toggleExpandFu(id) {
  const expandedRow = document.querySelector(`#tableBodyFu tr.expanded-row[data-parent="${id}"]`);
  const mainRow = document.querySelector(`#tableBodyFu tr.main-row[data-id="${id}"]`);
  if (!expandedRow) return;
  const isHidden = expandedRow.classList.contains('hidden');
  if (isHidden) {
    expandedRow.classList.remove('hidden');
    if (mainRow) mainRow.classList.add('expanded');
  } else {
    expandedRow.classList.add('hidden');
    if (mainRow) mainRow.classList.remove('expanded');
  }
}
```

**Verificação:** Clicar no ▼ expande/recolhe a linha correta na aba FU, sem afetar a aba Andamento.
