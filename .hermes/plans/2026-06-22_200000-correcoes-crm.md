## Task 3: Corrigir filtro de Empresa nas Métricas

**Objective:** O filtro "Caixa" / "Privados" nas métricas não filtrava processos da aba Análises porque eles usam campos booleanos (`banco_caixa`, `banco_bradesco`, etc.) e não o campo texto `banco`. Corrigir `metricaFiltrarPorEmpresa` para considerar ambos.

**Files:**
- Modify: `C:\Users\Prime04D\crm-uaicred\index.html` — função `metricaFiltrarPorEmpresa` (linhas 2792-2801)

### Step 1: Substituir a função metricaFiltrarPorEmpresa

**Localização:** index.html, linhas 2792-2801

**Código atual:**
```javascript
function metricaFiltrarPorEmpresa(lista) {
  if (metricaEmpresa === 'caixa') {
    return lista.filter(c => (c.banco || '').toLowerCase() === 'caixa');
  }
  if (metricaEmpresa === 'privados') {
    const privados = ['bradesco', 'itaú', 'itau', 'santander', 'inter'];
    return lista.filter(c => privados.includes((c.banco || '').toLowerCase()));
  }
  return lista;
}
```

**Novo código:**
```javascript
function metricaFiltrarPorEmpresa(lista) {
  if (metricaEmpresa === 'caixa') {
    return lista.filter(c => 
      (c.banco || '').toLowerCase() === 'caixa' || c.banco_caixa === true
    );
  }
  if (metricaEmpresa === 'privados') {
    return lista.filter(c => {
      const b = (c.banco || '').toLowerCase();
      return ['bradesco', 'itaú', 'itau', 'santander', 'inter'].includes(b) ||
        c.banco_bradesco === true || c.banco_itau === true || 
        c.banco_santander === true || c.banco_inter === true;
    });
  }
  return lista;
}
```
