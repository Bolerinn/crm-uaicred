**Step 2: Criar função `zerarBancoDeDados()` (JS)**

Adicionar após a função `excluirTodosLixeira()` (por volta da linha 3340). Função completa:

```javascript
async function zerarBancoDeDados() {
  if (!isMaster()) return;
  if (!confirm('⚠️ ATENÇÃO: Esta ação irá EXCLUIR PERMANENTEMENTE TODOS os clientes de TODAS as abas e TODOS os parceiros.\n\nEsta ação é IRREVERSÍVEL.\n\nDigite "ZERAR" para confirmar:')) return;
  
  const confirmacao = prompt('Digite ZERAR para confirmar a exclusão total:');
  if (confirmacao !== 'ZERAR') {
    mostrarToast('❌ Operação cancelada');
    return;
  }

  try {
    // 1. Deletar TODOS os clientes do Supabase (RLS permite só os do usuário)
    const { error } = await sb.from('clientes').delete().not('id', 'is', null);
    if (error) {
      mostrarToast('Erro ao zerar banco: ' + error.message);
      return;
    }

    // 2. Limpar parceiros do localStorage
    localStorage.removeItem('crm_indicacoes');

    // 3. Limpar array local e recarregar
    clientes = [];
    await carregarDados();
    popularDatalistIndicacoes();
    renderizarTudo();
    
    fecharConfig();
    mostrarToast('✅ Banco de dados zerado com sucesso. Todos os clientes e parceiros foram removidos.');
  } catch (e) {
    console.error('zerarBancoDeDados:', e);
    mostrarToast('❌ Erro inesperado ao zerar banco de dados');
  }
}
```

**Verificação:**
1. Abrir ⚙️ Configurações → ver seção "⚠️ Zona de Perigo" com botão vermelho
2. Clicar → confirmar → digitar "ZERAR" → deve limpar tudo
3. Verificar Aba 1: tabela vazia, datalist de parceiros vazio
4. Verificar modal "Ver parceiros": "Nenhum parceiro cadastrado"
5. Recarregar página → tudo permanece vazio (dados deletados do Supabase + localStorage)

### Item 2: Novo parceiro categorizado como "Outro" em vez da classe escolhida

**Root cause:** O `<select id="indTipo">` não é resetado ao fechar/abrir o modal. Se o usuário selecionou "Outro" na última vez, ele persiste como valor default na próxima abertura.

**Files:**
- Modify: `C:\Users\Prime04D\crm-uaicred\index.html` (2 pontos)

**Step 1: Resetar `indTipo` ao fechar o modal**

Linha 2515-2522, função `fecharModalIndicacao()`. Adicionar reset do select:

```javascript
function fecharModalIndicacao() {
  document.getElementById('modalIndicacao').classList.add('hidden');
  document.getElementById('indNome').value = '';
  document.getElementById('indTipo').value = 'Corretor';        // ← NOVA LINHA
  document.getElementById('indTipo').dispatchEvent(new Event('change')); // ← NOVA LINHA
  const imo = document.getElementById('indImobiliaria');
  if (imo) imo.value = '';
  const grp = document.getElementById('indImobiliariaGroup');
  if (grp) grp.classList.add('hidden');
}
```

**Step 2: Garantir que `abrirModalIndicacao()` também reseta**

Linha 2509-2513. Já despacha `change` via dispatchEvent, mas não reseta o valor. Adicionar reset ANTES do dispatch:

```javascript
function abrirModalIndicacao() {
  document.getElementById('modalIndicacao').classList.remove('hidden');
  document.getElementById('indTipo').value = 'Corretor';        // ← NOVA LINHA
  document.getElementById('indNome').focus();
  document.getElementById('indTipo').dispatchEvent(new Event('change'));
}
```

**Verificação:** Abrir modal → selecionar "Outro" → fechar → reabrir → o select deve mostrar "Corretor" (não "Outro").
